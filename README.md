### Run Locally

To run **CleverBooks** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/tarunnnrathoddd/CleverBooks.git
```

Windows:

```bash
git clone https://github.com/tarunnnrathoddd/CleverBooks.git
```
